# 고양이 장난감 등록 과제

## 고양이 장남감 CRUD 기능
  * 목록 : List
  * 상세 조회 : Detail
  * 등록 : Create
  * 수정 : Patch
  * 삭제 : Delete
  
##고양이 장난감 TDD 
  * BDD JUnit5을 적용
    
## 개발 순서  
  : 컨트롤롤러 개발 -> 서비스 -> 리포지토리 적용

## 계층형 구조 사용
* controller : 웹 계층 
* service : 비즈니스 로직, 트랜잭션 처리
* repository : DB 처리
* domain : 모든 계층에서 사용 가능
